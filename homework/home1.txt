Задача: Написать псевдокод, блок-схему или текстовое описание для 
алгоритма: сложить три числа и вывести сумму.
Реализовать алгоритм в коде.
 
Выполнение:
1. Псевдокод
Пусть num1 = 5
Пусть num2 = 10
Пусть num3 = 15

Пусть sum = num1 + num2 + num3

Вывести "Сумма: " + sum

2.Код в JS
const num1 = 5;
const num2 = 10;
const num3 = 15;

const sum = num1 + num2 + num3;

console.log("Сумма: " + sum);
